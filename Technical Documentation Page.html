Selectores

Un selector en CSS te permite elegir el elemento HTML al que le quieres dar estilo. Las categorías en las que se dividen los selectores son las siguientes:

  • Selectores simples: seleccionan el elemento con base en su id, clase o nombre.
  • Selectores combinados: seleccionan elementos basados en una relación específica entre ellos.
  • Pseudoclases: seleccionan elementos basados en un cierto estado.
  • Pseudoelementos: seleccionan una parte del elemento.
  • Selectores de atributo: seleccionan basados en un atributo o en el valor de un atributo.

Cuando 2 o más elementos comparten el mismo estilo pueden agruparse para ahorrar código. En el siguiente ejemplo el mismo estilo se aplicará a los elementos h1, h2 y p.

h1, h2, p {
text-align: center;
color: red;
}
Modelo Box

Todos los elementos del código HTML pueden ser considerados cajas y se usa el modelo box de CSS para estilizar dichas cajas. Cada caja tiene las siguientes partes:

  • Contenido: Zona donde el texto y las imágenes aparecen.
  • Padding: Espacio transparente alrededor del contenido.
  • Border: Espacio que rodea el contenido con su padding.
  • Margin: Línea que delimita la caja del resto de los elementos.

El ancho total de un elemento se calcula sumando el ancho de todas las partes de la caja. En el siguiente ejemplo el contenido tiene un ancho de 320px, el padding de 10 px (sumando padding derecho e izquierdo), el borde de 10px (sumando borde derecho e izquierdo) y el margen de 0px por lo que el ancho total de la caja del elemento div es de 350px.

div {
width: 320px;
padding: 10px;
border: 5px solid gray;
margin: 0;
}
Position

La propiedad de posición especifica el método de posicionamiento que se utiliza para un elemento. Existen 5 métodos de posicionamiento:

  • Static: Es el método por default, el elemento está siempre posicionado de acuerdo al flujo normal de la página.
  • Relative: El elemento está posicionado en relación a posición normal. Se pueden usar ajustar los valores top, left, right y bottom para mover el elemento desde su posición original. Otro contenido no se ajustará para entrar en el espacio dejado por este elemento.
  • Fixed: El elemento está posicionado en relación con la ventana gráfica lo que significa que permanece en la misma posición aún si la página se desplaza. Se pueden usar ajustar los valores top, left, right y bottom para mover el elemento.
  • Absolute: El elemento está posicionado en relación al antepasado más cercano, si no existe uno se posicionará en relación al cuerpo del html y se moverá junto a la pagina cuando ésta se desplaza. Estos elementos se remueven del flujo normal del documento por lo que pueden superponerse a otros elementos.
  • Sticky: El elemento está posicionado en relación a la posición del usuario cuando se desplaza por la ventana.

La posición se especifica como se muestra en el siguiente ejemplo donde el elemento div tiene una posición fija en la parte inferior de la ventana.

div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 300px;
border: 3px solid #73AD21;
}
Float

La propiedad float especifica como debe flotar un elemento, la propiedad clear espeifica cómo se comporta el elemento que rodea al elemento con la propiedad float. La propiedad float se utiliza para posicionar y dar formato al contenido y puede tener los siguientes valores:

  • left: el elemento flota a la izquierda de su contenedor.
  • right: el elemento flota a la derecha de su contenedor.
  • none: el elemento no flota, si no que se muestra tal y como se ubica en el texto.
  • inherit: el elemento hereda el valor de su elemento padre.

En el siguiente ejemplo, los 3 elementos div flotarán uno a lado del otro.

div {
float: left;
padding: 15px;
}
.div1 {
background: red;
}
.div2 {
background: yellow;
}
.div3 {
background: green;
}
Combinators

Son elementos que explican la relación entre dos elementos. Hay 4 diferentes:

  • Descendiente (espacio): Selecciona todos los elementos que descienden del elemento especificado.
  • Hijo (>): Selecciona todos los elementos hijos del elemento especificado.
  • Hermano adyacente (+): Selecciona el elemento adyacente a un elemento especificado
  • Hermano general (~): Selecciona todos los elementos hermanos que se encuentran a continuación del elemento especificado.

En el siguiente ejemplo todos los elementos p que se encuentren en seguida del elemento div tendran un fondo color amarillo.

div ~ p {
background-color: yellow;
}
